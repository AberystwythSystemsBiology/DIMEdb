Help Centre

This website contains information about how to interact with DIMEdb’s web application and API. Please ensure that you read it and check the FAQ’s before writing an email.

Frequently Asked Questions

Can I download all of DIMEdb?

Sure thing, just send us an email and we’ll get back to you as soon as we can(!)

What databases does DIMEdb use to collate its data from?

HMDB, PUBMED, CHEBI and KEGG.

The query I got is really large, how can I best view it?

If the output is a little large (> 200mb), it's probably not a good idea to open it all at once. The authors suggest using glogg to open up extremely large files.

API

DIMEdb features a application programming interface (API) which enables anyone to extract data without the need of the web application. The API is built with ease-of-use in mind, allowing you to download relevant information without having to download the entirety of the database.

Examples

Return all metabolites whose name contains "Ganglioside" 

{{ url }}api/metabolites/?name__contains=Ganglioside

Return all metabolites of which are found in "Food" 

{{ url }}api/metabolites/?origins__contains=Food

Return all metabolites that contain "Gangli" and are found in "Food" 

{{ url }}api/metabolites/?origins__contains=Food&name__contains=Gangli

Return all neutrally charged metabolites (M) that fall within a 10ppm range of 130m/z 

{{ url }}api/adducts/?adducts__pi_ppm=neutral,130,10

Return all negatively charged metabolites that can potentially be putatively ionised within a 20ppm range of 140/mz.

{{ url }}api/adducts/?adducts__pi_ppm=negative,140,20
